# wanandroid_flutter

玩 Android 是基于 Flutter 开发的跨平台的客户端应用，包括首页，项目，体系，搜索，妹子浏览，积分，主题切换，暗黑模式等功能。

项目地址： [https://github.com/xing16/WanAndroid-Flutter](https://github.com/xing16/WanAndroid-Flutter)
